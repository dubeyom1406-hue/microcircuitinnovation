# MicroCircuits Innovations

A modern, professional website for MicroCircuits Innovations Pvt. Ltd., showcasing expertise in semiconductor design, physical design, and design for testability services.

## Features

- **Responsive Design**: Fully responsive across all device sizes
- **Modern UI**: Sleek, professional interface with smooth animations
- **Interactive Elements**: Engaging user experience with framer-motion animations
- **Professional Sections**:
  - Home page with animated infinity logo
  - Expertise showcase (Physical Design, DFT, Verification)
  - About company with detailed information
  - Case Studies highlighting successful projects
  - Career opportunities with job listings
  - Contact form for inquiries

## Technology Stack

- **Frontend**: React.js with Vite
- **Styling**: Tailwind CSS and custom CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Routing**: React Router DOM

## Installation

1. Clone the repository:
```bash
git clone https://github.com/your-username/microcircuits.git
```

2. Navigate to the project directory:
```bash
cd microcircuits
```

3. Install dependencies:
```bash
cd microcircuits
npm install
```

4. Start the development server:
```bash
npm run dev
```
